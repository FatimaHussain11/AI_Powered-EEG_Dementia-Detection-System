The Project

About This Project

This system applies machine learning techniques to EEG recordings to assist in distinguishing between Alzheimer's Disease (AD), Frontotemporal Dementia (FTD), and Healthy Control (HC) subjects. It is designed as a decision-support aid, offering:

  • Non-invasive diagnosis based purely on EEG signal data
  • Faster screening compared to traditional clinical workflows
  • A dependable tool for academic and clinical research assistance
  • AI-assisted healthcare that supports, rather than replaces, clinicians
